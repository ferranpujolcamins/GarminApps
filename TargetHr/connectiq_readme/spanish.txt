Un campo de datos que muestra la frecuencia cardíaca objetivo de la fase de entrenamiento actual, en pulsaciones por minuto.

Cuando el objetivo es una zona de frecuencia cardíaca, el campo de datos muestra la frecuencia cardíaca que corresponde al punto medio de la zona.
Por ejemplo, si el objetivo actual es la zona 2, y tu zona 2 va de 130 a 150 pulsaciones por minuto, el campo de datos mostrará 140.

Cuando el objetivo actual está especificado con un límite inferior y un límite superior, el campo muestra la media de los dos valores.
Por ejemplo, si el objetivo es estar entre 140 y 180 pulsaciones por minuto, el campo de datos mostrará 160.

Cuando no hay ningún entrenamiento activo o el objetivo actual no es de frecuencia cardíaca, el campo de datos muestra "--".
